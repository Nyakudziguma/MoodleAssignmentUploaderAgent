Moodle Assignment Uploader
==========================

System Requirements:
- Windows 7/10/11 or Linux
- Google Chrome browser installed
- Internet connection

Instructions:
1. Double-click MoodleUploader.exe to launch the application
2. Configure your Moodle URL, username, password
3. Enter the Course ID and Assignment ID
4. Select the folder containing PDF files to upload
5. Click "Start Upload"

Note: Filenames should match student names (e.g., "John_Doe.pdf")
